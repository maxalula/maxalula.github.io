About the Game

This 3 game series was my first contact with programming and game design.

It is a series of platform games with engaging mechanics and enemies.

It was built using Macromedia Flash Player 8, and ActionScript as the scripting language.

The last game was released in october, 2009.


My Contribuitions

There was a lot of challenges during these projects, and much more learning than I could have ever imagined. In those I could grasp the understanting of object oriented programming, actions, events and input management.
